# Student Performance Analyzer

This project was developed as part of my Capstone Project for the Symbiosis Business Analyst Course (2026).
The Student Performance Analyzer is a data-driven academic analysis system designed to evaluate student performance based on grades and attendance data.

The project analyzes student academic performance and attendance data to:
- Classify performance levels
- Identify at-risk students
- Calculate subject-wise averages
- Provide actionable insights using Power BI

## Documents Included
- Business Requirements Document (BRD)
- Functional Specification Document (FSD)
- User Acceptance Testing (UAT)
- Final Project Report
- Power BI Dashboard

Tools Used:
- Python (pandas, matplotlib for EDA)
- Power BI
- Excel
-  SQL (Data Extraction & Querying)
